# Projekt Trello

Aplikacja do zarządzania zadaniami dla zespołów.

## Wymagania
- System logowania i rejestracji
- Tworzenie, edytowanie i usuwanie zadań
- Możliwość filtrowania i sortowania zadań

## Instalacja
1. Klonuj repozytorium:
git clone https://github.com/nazwa_uzytkownika/repo.git

2. Zainstaluj zależności:
npm install


## Testowanie
Testy dla aplikacji można znaleźć w folderze `test-cases`.
